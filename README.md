### 👥 如何使用

#### 第一步：创建账号

1. 点击右上角的 **"登录"** 按钮
2. 选择 **"注册"** 选项
3. 输入用户名和密码
4. 点击 **"注册"** 完成账号创建

#### 第二步：登录平台

1. 点击 **"登录"** 按钮
2. 输入用户名和密码
3. 点击 **"登录"** 进入平台
4. 系统会记住您的登录状态

#### 第三步：浏览演出

进入平台后，您可以看到：
- **🎬 播放列表**：展示所有可观看的演出
- 每个演出卡片显示标题、描述和封面
- 向下滚动查看更多演出

#### 第四步：观看演出

1. 点击您感兴趣的演出卡片
2. 进入独立播放页面
3. 点击播放按钮开始观看
4. 享受沉浸式的观看体验

#### ⌨️ 快捷键操作

在播放页面，您可以使用以下快捷键：

| 快捷键 | 功能 |
|--------|------|
| **空格** | 暂停/播放 |
| **← (左方向键)** | 快退 5 秒 |
| **→ (右方向键)** | 快进 5 秒 |
| **F** | 全屏 |
| **M** | 静音/取消静音 |

### ❓ 常见问题

#### Q1: 如果看到"账号在其他设备登录"的提示怎么办？

**A:** 这是我们的安全保护机制。为了保护您的账户安全，同一账号最多只能在一个设备上在线观看。

**解决方案：**
- 如果是您自己在另一台设备登录，可以在那台设备上登出
- 然后回到当前设备继续观看
- ⚠️ **小提示**：避免在同一浏览器中打开多个标签页同时观看，这可能会触发安全限制

#### Q2: 为什么视频加载不出来？

**A:** 这通常是网络连接问题。

**解决方案：**
1. 检查您的网络连接是否正常
2. 尝试刷新页面（按 F5 或 Ctrl+R）
3. 关闭其他正在下载的程序
4. 检查浏览器中是否有其他占用带宽的标签页
5. 如果问题持续，请稍后再试

#### Q3: 可以在手机上观看吗？

**A:** 当然可以！我们完全支持手机、平板等移动设备。

**建议：**
- 使用最新版本的浏览器以获得最佳体验
- 确保网络连接稳定（建议使用 Wi-Fi）
- 竖屏或横屏观看随意，界面会自动适配

#### Q4: 如何切换深色模式？

**A:** 非常简单！

- 点击右上角的 **🌙** (深色模式) 或 **☀️** (浅色模式) 图标
- 系统会记住您的选择
- 也会自动跟随您的系统设置

#### Q5: 忘记密码怎么办？

**A:** 请联系我们的客服获取帮助。

#### Q6: 去哪里购买节目？

**A:** 给[客服](mailto:support@neofantasy.online)发邮件，客服会告诉你的。
